## 🧰 Tech Stack

- **Backend:** Python, FastAPI, Pandas
- **Data:** NSE Bhavcopy CSV files
- **Frontend:** HTML, CSS, JavaScript, Chart.js
- **Docs:** Auto-generated Swagger UI

---

## 📊 Features

- Loads and cleans real NSE stock market data (Bhavcopy CSVs)
- Calculates key metrics for every stock:
  - Daily Return
  - 7-day Moving Average
  - 52-week High and Low
  - Volatility Score (custom metric)
- REST API with 3 endpoints
- Interactive dashboard with company list and price chart
- Search/filter companies in sidebar
- Auto-generated Swagger API docs

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/Anshuman56/stock-project
cd stock-project
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the data cleaning script
```bash
python clean.py
```
This generates `clean_data.csv` from the Bhavcopy files.

### 4. Start the API server
```bash
uvicorn main:app --reload
```

### 5. Open the dashboard
Go to: http://127.0.0.1:8000

## 📡 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/companies` | Returns list of all stock symbols |
| GET | `/data/{symbol}` | Returns last 30 days of stock data |
| GET | `/summary/{symbol}` | Returns 52W high, low, avg close |
| GET | `/docs` | Interactive Swagger API documentation |

### Example Responses

**GET /summary/TCS**
```json
{
  "symbol": "TCS",
  "52w_high": 3890.0,
  "52w_low": 3541.0,
  "avg_close": 3721.5
}
```

## 📐 Data Metrics Explained

| Metric | Formula |
|--------|---------|
| Daily Return | (Close - Open) / Open |
| 7-day MA | Rolling 7-day average of Close price |
| 52W High | Maximum High price across all data |
| 52W Low | Minimum Low price across all data |
| Volatility Score | Standard deviation of Daily Returns |

---

## 💡 Custom Metric — Volatility Score

Each stock is assigned a **Volatility Score** based on the standard deviation
of its daily returns. A higher score means the stock price moves more unpredictably.
This can help identify high-risk vs stable stocks at a glance.
